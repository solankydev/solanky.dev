Animated GIFs are not animating in
Internet Explorer 7

Today I have placed a GIF image in one web
page, it was displaying fine in Firefox and Internet Explorer 6 but
there was no animation in Internet Explorer 7. Then I have searched
about it, the solution is very easy. By default Internet Explorer 7
has disabled animations in web page. To enable it go to **Tools
\> Internet Options > Advanced** and check the **“Play
animations in web pages”** in **Multimedia** section.

Problem solved…..
