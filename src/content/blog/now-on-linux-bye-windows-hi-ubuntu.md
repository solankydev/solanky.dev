I am a .NET programmer so by default I use Windows both at my work and home. I was always willing to use Linux but it was a very hard decision to take because of .NET. But last week I have purchased my new laptop and I have decided to install Linux on it instead of Windows. I have heard a lot about Ubuntu so I have downloaded the latest version 7.04 (Feisty Fawn), burn the download to CD and start the installation. Once I have seen the Ubuntu installation process so I have not faced any problem. In just around half an hour my laptop was loaded with Ubuntu.
Now after one week of experience with Ubuntu I can say one thing – I love it. Before migrating to Ubuntu I was in doubt that I will face some problems using Linux but I was wrong. Now I am more happy than earlier while using Windows.
Still I have to use Windows at work because of (as I already have said) .NET but I think now I will never go back to Windows at home at least. Only time will tell about the future but I am seriously thinking to migrate to Java/Open Source now from .NET.

I will be writing about my Ubuntu experience in future articles.
