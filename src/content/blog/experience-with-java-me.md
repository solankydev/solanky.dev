I have tried my best to keep my promise of regular blogging but could not, as usual. It is becoming very difficult to find time for doing so. I wrote my last post when I got a new assignment for developing a Java ME application, and now I am happy to write that it has been completed. It was a nice experience of getting hands dirty in Java ME, Bluetooth API (JSR82), RFCOMM protocol, btspp, etc. Now after 4 months of experience it feels very good that we are able to develop a Java ME application in house. And this feels even better when I know that we are a team working only in the .NET platform so far.
