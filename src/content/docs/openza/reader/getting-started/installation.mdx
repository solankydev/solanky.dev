## Install from Microsoft Store

[Install Openza Reader from the Microsoft Store](https://apps.microsoft.com/detail/9NNPMN0JSSW5?hl=en-us&gl=IN)

The Microsoft Store installs updates automatically and lets you uninstall Reader from Windows Settings.

## Requirements

- Windows 10 version 22H2 or later, or Windows 11
- A 64-bit Windows PC

After installation, open **Openza Reader** from the Start menu. You can also open a Markdown file from File Explorer and choose Openza Reader.
