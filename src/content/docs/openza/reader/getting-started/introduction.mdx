---

Openza Reader opens `.md` and `.markdown` files on Windows without changing them.

## What you can do

- **Open files.** Open a file from the app, File Explorer, or drag and drop.
- **Switch views.** Read the rendered document, view the raw Markdown, or show both side by side.
- **Navigate and search.** Use the table of contents, text search, recent files, and zoom controls.
- **Reload changes.** Reload a file after another app changes it while keeping your approximate position.

Reader is read-only. It does not edit Markdown files.
