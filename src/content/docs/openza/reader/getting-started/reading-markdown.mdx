---
title: Reading Markdown
description: Open and read Markdown files in Openza Reader
---

Openza Reader is designed around a simple reading workflow.

## Open a file

You can open Markdown files by:

- Selecting **Open** in Reader.
- Dragging a `.md` or `.markdown` file into the window.
- Opening a Markdown file from File Explorer after installing Reader.

## Navigate

Reader builds a table of contents from the document headings. Select a heading to jump to that section.

## Choose a view

- **Preview** shows the formatted document.
- **Raw** shows the original Markdown as read-only text.
- **Side by side** shows both views together.

## Search

Select **Search** or press `Ctrl+F` to find text in the document.

## Zoom and copy

Use the zoom controls to change the reading size. Select text and copy it from the toolbar or with a keyboard shortcut.

## Open in an editor

Reader does not change Markdown files. Select **Editor** when you want Windows to open the current file in an installed editing app.

## Settings and recent files

Settings include the default view, reader theme, remote-image preference, document information, and recent-file controls. Your recent-file list stays on this PC.
