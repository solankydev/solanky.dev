---
title: Markdown Support
description: Markdown features supported by Openza Reader
---

Reader supports the Markdown elements commonly used in notes and documentation.

## Supported content

- Headings
- Lists
- Tables
- Task lists
- Fenced code blocks
- Blockquotes
- Links
- Images stored with the document
- Optional images from the web

## Code blocks

Reader adds syntax highlighting to many fenced code blocks. The required highlighting files are included with the app, so code blocks do not depend on an online service.

## HTML in Markdown files

Reader does not run HTML included in a Markdown file. HTML-like text is shown as text instead. This keeps documents predictable and reduces the risk of opening an unfamiliar file.

## Reader themes

Choose System, Light, Dark, or Sepia in Reader settings. Windows continues to control High Contrast colors.
