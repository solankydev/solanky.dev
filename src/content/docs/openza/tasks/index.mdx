## What you can do

- **Keep tasks on your device.** Use Tasks without creating an account or connecting another service.
- **Organize tasks.** Use projects, labels, dates, priorities, notes, and saved views.
- **Connect other services.** Add Todoist or Microsoft To Do if you want task sync.
- **Back up your data.** Create local backups and optionally keep backup snapshots in OneDrive.
- **Use Markdown.** Import Markdown checkboxes and export tasks as Markdown files.
