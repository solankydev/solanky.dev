## Install from Microsoft Store

[Install Openza Tasks from Microsoft Store](https://apps.microsoft.com/detail/9NQGDKXGRGF8)

You can also [open the listing in the Microsoft Store app](ms-windows-store://pdp/?ProductId=9NQGDKXGRGF8).

The Microsoft Store installs updates automatically and lets you uninstall Tasks from Windows Settings.

## Requirements

- Windows 10 version 22H2 or later, or Windows 11
- A 64-bit Windows PC

After installation, open **Openza Tasks** from the Start menu. You can begin using local tasks without signing in.
