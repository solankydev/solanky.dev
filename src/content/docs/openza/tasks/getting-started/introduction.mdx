title: Openza Tasks
description: What Openza Tasks does
---

Openza Tasks keeps tasks, projects, and labels on your Windows device. You can use it by itself or connect Todoist and Microsoft To Do.

## What you can do

- **Keep tasks locally.** Start without an account. Your tasks remain available on your device.
- **Organize your work.** Use projects, labels, dates, priorities, notes, and filters.
- **Connect other services.** Add Todoist or Microsoft To Do only if you want task sync.
- **Back up your data.** Create local backups and optionally keep backup snapshots in OneDrive.
- **Import and export Markdown.** Import Markdown checkboxes and export tasks grouped by project.

## Get started

1. [Install Tasks](/openza/tasks/getting-started/installation).
2. Add tasks and projects.
3. [Connect another service](/openza/tasks/getting-started/configuration) only if you want sync.
4. Set up backups for recovery.
