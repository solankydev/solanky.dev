Openza Tasks works without an account. Connect other services only when you want to sync tasks or keep backups in OneDrive.

## Available connections

| Feature | Availability | How to connect |
|---------|--------------|----------------|
| Tasks stored in Openza | Available | No sign-in required |
| Todoist | Available | Todoist access token |
| Microsoft To Do | Available | Microsoft sign-in |
| OneDrive backup | Optional | Microsoft sign-in |

## Start without a connection

You can begin adding tasks immediately. Your tasks remain on your device unless you choose to connect another service.

To connect Todoist or Microsoft To Do, open **Settings > Integrations**. To keep backup copies in OneDrive, open **Settings > Backups**.

## Connection guides

- [Connect Todoist](/openza/tasks/guides/todoist-setup)
- [Connect Microsoft To Do](/openza/tasks/guides/mstodo-setup)

## Settings

Open Settings from the left navigation:

- **Appearance** — Choose a light or dark theme.
- **Integrations** — Connect or disconnect Todoist and Microsoft To Do.
- **Spaces** — Keep separate sets of projects and tasks.
- **Backups** — Manage backups on this PC, optional OneDrive backups, and restore options.

Import and export actions are available from the task command bar.

<Aside type="note">
Microsoft To Do and OneDrive backup can use the same Microsoft account or different accounts.
</Aside>
