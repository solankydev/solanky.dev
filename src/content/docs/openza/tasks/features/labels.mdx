title: Labels
description: Categorize tasks with labels
---

Labels help you categorize tasks across projects. Todoist calls them labels, while Microsoft To Do uses categories.

## Viewing Labels

Access labels through:
- **Next Actions view** - Shows active tasks with at least one label
- **Task details** - See all labels applied to a task
- **Label filter** - Filter the current task list by a specific label
- **Grouping** - Group task lists by label when the current view supports grouping

## Label Colors

Labels and categories from connected services are brought into Openza when available:

- Todoist labels are imported as labels
- Microsoft To Do categories are imported as labels when available
- Changes you make to labels in Openza stay in Openza

## Filtering by Label

The label dropdown above the task list shows all available labels from your tasks. Select a label to show only tasks with that label applied.

## Flexible Workflow

Use labels for any dimension that matters to you, such as location, person, area, energy, or review bucket. Choose labels that match the way you work.

See [Projects](/openza/tasks/features/projects) for organizing tasks into project groups.
