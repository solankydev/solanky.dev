Projects group related tasks inside the selected Openza space.

## Projects in Openza

Projects created in Openza stay on your device:

- Create, edit, and delete freely
- Move tasks between them
- Use them to filter the current task list

## Projects from connected services

Tasks keeps the original Todoist project or Microsoft To Do list name so you can see where a connected task came from:

- The original name appears with the connected task
- You can still assign the task to any Openza project
- Manage Todoist projects and Microsoft To Do lists in their original apps

<Aside type="note">
When you add a connected task to Inbox, you can assign it to any Openza project.
</Aside>

### The Inbox

Inbox is the default place for unprojected tasks:

- Tasks without a project go here
- Newly found items from connected services can be reviewed first
- Select a project later from task details

## Creating a Project

<Steps>
1. Click **New project** in the command bar
2. Enter a project name
3. Click **Create**
</Steps>

The new project appears in the Projects pane for the selected space.

## Editing a Project

<Steps>
1. Find the project in the Projects pane
2. Click **Edit**
3. Change project details supported by the current app
4. Click **Save**
</Steps>

<Aside type="note">
Edit Todoist project names or Microsoft To Do list names in their original apps, then sync.
</Aside>

## Deleting a Project

<Steps>
1. Select the project
2. Click **Delete**
3. Confirm the deletion
</Steps>

<Aside type="caution">
Deleting a project cannot be undone. Move tasks to another project or Inbox first if you want to keep the project grouping.
</Aside>

## Moving Tasks Between Projects

You can move tasks to a different project in two ways:

### From Task Details

<Steps>
1. Select a task to open the details pane
2. Use the project dropdown to select a new project
3. Click **Save task**
</Steps>

## Filtering by Project

Click any project in the Projects pane to filter the task list:

- Only tasks from that project are shown
- The project name appears in the header
- Select **All projects** or another view to clear the filter

## Spaces

Projects belong to the selected Openza space. A space keeps one set of projects and tasks separate from another. Manage spaces in **Settings > Spaces**.
