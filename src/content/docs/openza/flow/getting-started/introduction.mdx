title: Openza Flow
description: What Openza Flow does
---

Openza Flow is a native Windows dashboard for GitHub activity. You can see review requests, pull requests you created, recent repository releases, and GitHub Actions workflow runs.

## What you can do

- **See review requests.** View pull requests where your review has been requested.
- **Follow your pull requests.** See pull requests you created and open them on GitHub.
- **Monitor repository activity.** Check recent releases and Actions workflow runs across repositories in the selected organization.
- **Search and filter.** Search pull requests and filter them by organization.
- **Choose background updates.** Turn on tray mode and notifications only if you want Flow to keep checking.

## Get started

1. [Install Flow](/openza/flow/getting-started/installation).
2. [Sign in with GitHub](/openza/flow/getting-started/github-token).
3. Open Review Requests, Created PRs, Releases, or Actions.

Flow provides read-only views of GitHub activity. It opens GitHub when you want to act on a pull request, release, or workflow run.
