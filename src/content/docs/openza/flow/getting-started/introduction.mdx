title: Openza Flow
description: What Openza Flow does
---

Openza Flow shows GitHub pull requests in a Windows app. You can see review requests, pull requests you created, recently reviewed work, and recently created work.

## What you can do

- **See review requests.** View pull requests where your review has been requested.
- **Follow your pull requests.** See pull requests you created and open them on GitHub.
- **Search and filter.** Search pull requests and filter them by organization.
- **Choose background updates.** Turn on tray mode and notifications only if you want Flow to keep checking.

## Get started

1. [Install Flow](/openza/flow/getting-started/installation).
2. [Sign in with GitHub](/openza/flow/getting-started/github-token).
3. Open Review Requests or Created PRs.

Flow does not approve, comment on, merge, or request changes. It opens GitHub when you want to act on a pull request.
