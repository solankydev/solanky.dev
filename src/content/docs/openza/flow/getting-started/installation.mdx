title: Installation
description: Availability of Openza Flow for Windows
---

Openza Flow is being prepared for its first native Windows release. It is not yet available in the Microsoft Store.

## Requirements

- Windows 10 version 22H2 or later, or Windows 11
- A 64-bit Windows PC
- A GitHub account

## Current availability

Check [GitHub Releases](https://github.com/openza/flow/releases) for available downloads and release notes. Older releases may belong to the previous version of Flow, so read the release description before installing.

The Microsoft Store will become the recommended installation and update method when the native Windows release is ready.
