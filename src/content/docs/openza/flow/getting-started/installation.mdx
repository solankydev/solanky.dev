## Install from Microsoft Store

[Install Openza Flow from the Microsoft Store](https://apps.microsoft.com/detail/9N37DN0X68PZ)

You can also [open the listing in the Microsoft Store app](ms-windows-store://pdp/?ProductId=9N37DN0X68PZ).

The Microsoft Store installs updates automatically and lets you uninstall Flow from Windows Settings.

## Requirements

- Windows 10 version 22H2 or later, or Windows 11
- A 64-bit Windows PC
- A GitHub account

After installation, open **Openza Flow** from the Start menu and [sign in with GitHub](/openza/flow/getting-started/github-token).

## Release notes

Openza Flow v1.0.0 is the first stable native Windows release. See the [v1.0.0 release notes](https://github.com/openza/flow/releases/tag/v1.0.0) for highlights and changes.
