## What you can do

- **See review requests.** View pull requests where someone has requested your review.
- **Follow your pull requests.** See pull requests you created and open them on GitHub.
- **Monitor releases and Actions.** See recent repository releases and GitHub Actions workflow runs.
- **Search and filter.** Search pull requests and filter them by organization.
- **Choose notifications.** Turn on background refresh and Windows notifications if you want them.
