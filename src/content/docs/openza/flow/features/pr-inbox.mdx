---
title: Pull Request Dashboard
description: Understand the Openza Flow dashboard
---

The dashboard groups pull request work into focused views so you can scan quickly and open GitHub only when you need to act.

## Review requests

Shows open pull requests where your review is requested. This is the default view.

## Created pull requests

Shows open pull requests created by you. Use it to follow feedback and check whether anything needs a response.

## Recent activity

Recently Reviewed and Recently Created keep recent pull requests within reach after they leave the main views.

## Search and organization filter

Search by repository, author, or title. Choose an organization when you want to separate work, community, and personal repositories. Flow remembers the selected organization.

## Releases and workflow runs

The Releases and Actions pages show recent activity from repositories in the selected organization. Flow checks up to 50 recently updated repositories. Draft releases appear only when your GitHub account can view them.

## Recent information

Flow keeps the first page of recent information on this PC so the dashboard can appear quickly when reopened. Search results and organization-specific views are refreshed from GitHub.
