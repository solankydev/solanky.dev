## Development

When starting the dev server, use background mode:

```
astro dev --background
```

Manage the background server with `astro dev stop`, `astro dev status`, and `astro dev logs`.

## Public repository hygiene

This is a public repository. Never commit subscriber exports, email addresses, analytics exports, API keys, Cloudflare credentials, Beehiiv credentials, private drafts, or unreviewed personal media. Run `gitleaks detect --source . --verbose` before commit-readiness checks.
