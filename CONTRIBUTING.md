# Contributing

This repository is published for transparency and deployment of [solanky.dev](https://solanky.dev).

I am not accepting external contributions, pull requests, feature requests, or content edits for this website repository.

The Openza product source code lives in the respective product repositories:

- [openza/tasks](https://github.com/openza/tasks)
- [openza/reader](https://github.com/openza/reader)
- [openza/flow](https://github.com/openza/flow)

If you notice a security issue, please report it privately as described in [SECURITY.md](./SECURITY.md).
